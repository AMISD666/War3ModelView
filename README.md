# 咕咕 War3 模型编辑器（War3ModelView）

基于 **Warcraft III (War3) 模型格式（MDX/MDL）** 的可视化查看与编辑工具，集成模型预览（Viewer）与多类编辑器面板，支持常见资源（贴图/材质/动画/节点/粒子/丝带等）修改，并提供一系列模型维护与批处理工具。

---

## 功能一览

### 模型文件与导入导出

- **导入模型**：打开 `MDX` / `MDL` 文件
- **最近文件**：最近打开记录、清空历史
- **保存模型**：原路径保存
- **复制模型**：复制当前模型数据（用于合并/对比等工作流）
- **一键 MDL/MDX 互转**
- **另存为 MDL / 另存为 MDX**
- **TXT 打开**：以文本视图打开/编辑模型文本（用于快速检视与修复）

### 编辑器与管理器（面板/弹窗）

- **节点管理器**：节点树浏览、编辑常见节点属性（含独立窗口编辑链路）
- **镜头管理器**：Camera 管理
- **多边形管理器**：Geoset 管理与相关编辑
- **多边形动画管理器**：GeosetAnim 管理与编辑
- **贴图管理器 / 贴图编辑器**
- **贴图动画管理器**：TextureAnim 管理与编辑
- **材质管理器 / 材质编辑器**
- **动作管理器**：Sequence 管理与编辑
- **全局序列管理器**：Global Sequence 管理
- **快捷键设置**：快捷键配置与查看

### Viewer 预览与模式

- **模式切换**：
  - 查看模式
  - 顶点模式
  - UV 模式
  - 动画模式
- **显示开关**：网格、节点、骨架、碰撞体、镜头、灯光、挂接点等（以实际 UI 为准）
- **渲染模式**：贴图/线框等
- **背景色、FPS 显示、调试控制台**

### 工具与批处理

- **重新计算法线**
- **重新计算模型范围（Extents）**
- **修改模型位置/旋转/缩放**
- **模型优化**：清理/压缩/修复类操作集合（以实际弹窗为准）
- **模型合并**
- **多边形显隐工具**
- **消散动画工具**
- **添加死亡动画**
- **删除所有灯光**
- **修复模型**
- **合并相同材质**
- **清理未使用材质 / 清理未使用贴图**

### MPQ 资源浏览

- **MPQ 浏览**：浏览 Warcraft III MPQ 资源包内容（贴图等资源定位与查看）

---

## 适用场景

- **MDX/MDL 模型的快速预览与检查**：几何体、贴图、动画、节点层级等
- **模型资源维护**：贴图/材质整理、未使用资源清理、材质合并
- **动画与可见性相关编辑**：动作（Sequence）、全局序列、GeosetAnim、贴图动画等
- **常见修复与优化**：Extents/法线重算、模型修复、灯光清理等
- **从 MPQ 中查找资源**：辅助定位模型引用贴图等资源

## 环境要求（Windows）


- **Node.js**：建议使用 LTS（18/20 均可，项目当前 `@types/node` 为 18 系列）
- **Rust**：建议安装稳定版（stable）工具链
- **Tauri 2 依赖**：
  - Windows 构建工具链（Visual Studio Build Tools，包含 C++ 构建组件）
  - WebView2 Runtime（多数 Win10/11 默认已安装；若打包时提示缺失可手动安装）

---

## 安装与运行（开发）

在仓库根目录执行：

```bash
npm install
```

### 仅启动前端（Vite）

```bash
npm run dev
```

默认前端地址为 `http://localhost:5173`（见 `src-tauri/tauri.conf.json` 的 `devUrl`）。

### 启动桌面应用（Tauri）

```bash
npm run tauri dev
```

---

## 构建与发布

### 构建前端产物

```bash
npm run build
```

### 打包桌面安装包（Tauri）

```bash
npm run tauri build
```

Windows 目标包含 `nsis` 安装包配置（见 `src-tauri/tauri.conf.json`）。

---


## 目录结构（核心）

- `src/renderer/`：前端（React/Vite）主工程
- `src-tauri/`：Tauri 2 桌面壳（Rust）
