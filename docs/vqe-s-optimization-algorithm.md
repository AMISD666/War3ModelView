# 模型多边形优化升级方案：VQE-S (Visual Quadric Error - Skinned)

## 1. 方案背景
当前的多边形优化方案基于“自适应五档边折叠”启发式算法。虽然通过多轮不同强度的 Pass 达到了压缩目的，但存在以下核心痛点：
- **误差衡量维度单一**：仅考虑点距与 UV 距，未考虑曲率（法线）的累积误差。
- **全局非最优**：按分档串行处理，无法保证优先折叠整个网格中代价最小的边。
- **蒙皮相关性弱**：对骨骼权重的保护主要依赖签名匹配，在高压缩比下易导致蒙皮拉伸变形。

## 2. 超思维分析 (Ultrathink Reasoning)
### 2.1 核心数学模型：8D 扩展二次误差矩阵 (QEM)
我们将标准的 Garland-Heckbert QEM 扩展至 **8维空间**：
- **Geometry (3D)**: $P(x, y, z)$ 保证体积与轮廓。
- **Texture (2D)**: $UV(u, v)$ 保证纹理映射不拉伸。
- **Skinning (3D)**: $S(b_1, b_2, b_{main})$ 保证骨骼权重的语义一致性。

每个顶点 $v$ 维护一个 $9 \times 9$ 的对称矩阵 $Q$。折叠边 $(v_i, v_j)$ 的代价函数由 $Q' = Q_i + Q_j$ 决定，通过求解 $Q' v_{new} = 0$ 得到数学意义上的**最优坍缩位置**。

### 2.2 全局优先级队列 (Global Priority Queue)
废弃现有的“多档 Pass”机制，采用全局最优寻找策略：
- **初始化**：计算所有合规候选边的 QEM 代价，推入最小堆（Min-Heap）。
- **迭代**：每次取出代价最小的边执行折叠，并动态更新邻接边的代价。
- **收敛**：直到达到目标面数或最小代价超过安全阈值。

### 2.3 语义与结构保护 (Visual & Structural Safety)
- **Feature Edge Protection**：对边界边、材质分割线施加高权重惩罚项。
- **Flip-Prevention**：局部法线翻转校验，预防自交。
- **Bone Influence Consistency**：计算蒙皮权重差异，差异过大则赋予极高代价，防止绑定崩坏。

---

## 3. 方案优势
- **数学完备**：QEM 提供了严谨的网格简化误差界限，优化的平衡性远超启发式算法。
- **细节保留**：8D 空间模型确保了纹理细节（UV）和骨骼绑定（Skinning）在减面过程中协同优化，不再是割裂的约束。
- **极致压缩**：由于始终处理全局代价最小的边，在达到相同面数时，VQE-S 的视觉质量显著优于现有的分档 Pass 方案。

---

## 4. 参数建议
- **保形权重 (Lambda)**：调节几何与 UV/蒙皮之间的权重分配。
- **边界锁定 (Boundary Lock)**：对于需要完美拼接的模型，可强制锁定边界边。
