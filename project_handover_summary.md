# Warcraft 3 Model Editor - 项目进度交接文档

## 1. 项目概况
- **类型**: 基于 Tauri + React + TypeScript 的桌面应用程序。
- **目标**: 开发一个独立的魔兽争霸3模型 (.mdx/.mdl) 查看器和编辑器。
- **核心库**: 使用本地的 `@war3-model` 库处理 WebGL/WebGPU 渲染和模型解析。

## 2. 当前进度与已完成功能

### 核心功能
- **模型加载**:
  - 支持 `.mdx` (二进制) 和 `.mdl` (文本) 格式。
  - 支持拖拽加载或通过菜单打开。
- **纹理加载 (已修复/增强)**:
  - **策略**: 优先尝试从 MPQ 档案加载 (如 `Textures\`, `UI\` 等路径)，失败后尝试本地文件系统。
  - **路径处理**: 自动处理 `war3mapImported\` 前缀，支持多种本地路径搜索策略 (当前目录、文件名匹配、向上递归查找)。
- **MPQ 支持**:
  - 支持加载魔兽争霸3的 `.mpq` 档案。
  - 自动保存已加载的 MPQ 路径到 `localStorage`，下次启动自动加载。

### 渲染系统
- **WebGL 环境**:
  - 实现了 WebGL2 上下文初始化。
  - **修复**: 解决了显示骨架/节点时模型消失的问题 (在每帧开始时重置 WebGL 状态 `DEPTH_TEST`, `BLEND`)。
  - **修复**: 解决了滚轮拉远后模型消失的问题 (将远裁剪面 Far Clipping Plane 从 2000 增加到 20000)。
- **显示选项**:
  - 网格 (Grid)、节点 (Nodes)、骨架 (Skeleton)、线框模式 (Wireframe) 可切换。
  - 队伍颜色 (Team Color) 和 队伍光晕 (Team Glow) 支持。
  - 背景颜色可自定义。

### 摄像机控制 (最新增强)
- **操作方式**:
  - **左键拖拽**: 旋转视角 (Orbit)。
  - **右键/中键拖拽**: 平移视角 (Pan)。
  - **滚轮**: 缩放 (Zoom)。
- **视角预设**:
  - **快捷键**: 数字键 `1`-`6` 对应 前/后/左/右/顶/底 视图。
  - **聚焦**: `F` 键重置视角并聚焦中心。
  - **菜单**: 顶部菜单栏新增 "视图" (View) 菜单，包含上述所有功能。
- **注意**: 曾实现了平滑插值 (Smooth Interpolation)，但因用户体验原因**已移除**，目前为即时响应模式。

### UI/UX
- **布局**: 左侧动画列表，中间主视口，顶部菜单栏。
- **设置持久化**: 所有显示设置 (网格、FPS、背景色等) 均保存至 `localStorage`。
- **修复**: 修复了背景颜色选择器点击时意外关闭菜单的问题。

## 3. 关键文件说明

- **`src/renderer/src/components/Viewer.tsx`**
  - **核心组件**: 负责 3D 渲染循环、摄像机逻辑、输入事件处理。
  - **状态**: 维护 `targetCamera` (摄像机位置)、`renderer` (模型渲染器实例)。
  - **关键逻辑**: `loadModel` (纹理加载策略), `render` (渲染循环), `handleMouseMove` (摄像机控制)。

- **`src/renderer/src/components/MainLayout.tsx`**
  - **主布局**: 管理全局状态 (modelPath, animations, settings)。
  - **通信**: 处理 `MenuBar` 和 `Viewer` 之间的状态传递 (如 `viewPreset`)。
  - **MPQ**: 处理 MPQ 文件的加载和自动恢复。

- **`src/renderer/src/components/MenuBar.tsx`**
  - **菜单组件**: 包含文件、视图、设置菜单。
  - **交互**: 通过回调函数控制全局状态。
